PROJECTS

  1. Instagram Clone

    An Instagram clone implemented with client-side dynamic pages, REST APIs, and React. Has all basic functionalities of Instagram including creating an account, following other users, creating, liking, commenting on, and deleting posts. User authorization implemented with HTTP Basic Access Authentication and Flask. Deployed on AWS for testing.

    • Python
    • Flask
    • React
    • REST API
    • HTML/CSS
    • JavaScript
    • SQLite3
    • AWS

  2. Search Engine

    A scalable dynamic search engine similar to Google implemented with an index server, a search server, and a REST API app. The search engine uses both text-analysis (tf-idf) and link analysis (PageRank), and allows the user to decide the PageRank weight. The inverted index of each HTML document is done through a 5-stage MapReduce pipeline. Deployed on AWS for testing.

    • Python
    • Flask
    • REST API
    • HTML/CSS
    • JavaScript
    • SQLite3
    • BeautifulSoup
    • AWS

  3. epiFoods

    An iOS app that helps users find food that is safe for their allergies. The app allows users to search for food by name, scan barcodes, and view detailed information about each food item. It also provides a personalized experience by allowing users to save their favorite foods and allergies.

    • Python
    • Flask
    • REST API
    • HTML/CSS
    • JavaScript
    • SQLite3
    • BeautifulSoup
    • AWS
